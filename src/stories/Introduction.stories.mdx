# Table Component

The Table component is a custom React component that provides a way to display tabular data in a responsive, sortable, and selectable format. 
The component is composed of several sub-components, including TableHeader and TableBody, which are used to display the header and body of the table, respectively.

<div className="subheading">Functionality</div>

The component also includes a sort feature, which allows users to sort the table data by clicking on the header of the column they want to sort by. The sort order can be changed by clicking on the header again.

The Table component is fully responsive and will adjust its display to fit the available space on the screen, whether viewed on a desktop or mobile device.

Overall, the Table component provides a powerful and flexible way to display tabular data, making it a valuable tool for any project that requires the display of structured data.
